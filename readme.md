# CourseTalk

## About
[CourseTalk](https://course-talk.herokuapp.com) is an online platform for student collaboration. Created for [HackPrinceton](https://hackprinceton.com) Fall 2018 ([Devpost](https://devpost.com/software/coursetalk)).
